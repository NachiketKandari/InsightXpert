# Team Execution Plan

**Q: Briefly describe how your team plans to divide work and execute the solution if shortlisted.**

## Team (2 members)

- **Nachiket (Lead — System design + AI pipeline):** Owns overall system architecture, UI/UX design, and the full LLM pipeline — model selection, prompt engineering, query understanding, SQL generation, response synthesis, validation layers, routing logic, and evaluating local/open-source model alternatives. Also responsible for end-to-end integration and technical decision-making.
- **Arush (Data layer + Infrastructure):** Owns the data layer implementation, SQL execution engine, API endpoints, and deployment infrastructure. Handles dataset exploration, feature engineering, and builds the analytical functions (aggregations, trend detection, comparisons) exposed as internal APIs.

Both collaborate on system design decisions and scope definition upfront.
